# flickr-react
A Flickr search implementation built with React.

## Installation
First, grab the repo:
```bash
git clone https://github.com/OvestLabs/flickr-react.git
```
Then either build with NPM:
```bash
npm install
npm start
```
or Yarn:
```bash
yarn install
yarn start
```
Now point your browser to
```
http://localhost:8080
```
